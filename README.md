=======

# JavaScriptWebHtml

Paprastas statinis interneto svetainės projektas, skirtas pet sitting paslaugų pristatymui. Projekte naudojami HTML puslapiai, atskiri vidiniai puslapiai ir statiniai resursai.

## Projekto struktūra

```
JavaScriptWebHtml/
├── index.html
├── IMG_2524.HEIC
├── privatumo-politika.html
├── salygos.html
├── assets/
│   └── img/
│       └── happy_pets.jpg
└── pages/
    ├── index.html
    ├── naujienos.html
    ├── kainos.html
    ├── kontaktai.html
    ├── rezervacija.html
    ├── paslaugos.html
    └── kaipTaiVeikia.html
```

### Projekto struktūros diagrama

![Projekto struktūra](./docs/projekto-struktura.png)

## Puslapių aprašymas

- `index.html` – pagrindinis svetainės puslapis
- `privatumo-politika.html` – privatumo politikos informacija
- `salygos.html` – taisyklės ir sąlygos
- `pages/index.html` – vidinis puslapis arba papildoma pagrindinio puslapio versija
- `pages/naujienos.html` – naujienų puslapis
- `pages/kainos.html` – kainų puslapis
- `pages/kontaktai.html` – kontaktų puslapis
- `pages/rezervacija.html` – rezervacijos puslapis
- `pages/paslaugos.html` – paslaugų puslapis
- `pages/kaipTaiVeikia.html` – proceso paaiškinimo puslapis
- `assets/img/` – paveikslėliai ir kita vizualinė medžiaga

## Naudojamos technologijos

- HTML5
- CSS3
- Statiniai vaizdai (`assets/img/`)

## Projekto paleidimas

Kadangi tai statinis projektas, jį galima atidaryti tiesiog naršyklėje:

1. Atidarykite `index.html` failą naršyklėje

arba

2. Paleiskite projektą per lokalų serverį, pavyzdžiui naudojant VS Code `Live Server`
